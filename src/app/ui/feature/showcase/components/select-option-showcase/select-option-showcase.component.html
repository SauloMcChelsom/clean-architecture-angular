SelectOption

O SelectOption é um componente de interface do usuário (UI) que permite ao usuário selecionar uma opção entre várias alternativas. Ele é comumente usado em formulários, menus e listas para fornecer uma maneira fácil e intuitiva de escolher uma opção. O Select-Option pode ser apresentado em diferentes formatos, como uma lista suspensa, botões de opção ou caixas de seleção.
