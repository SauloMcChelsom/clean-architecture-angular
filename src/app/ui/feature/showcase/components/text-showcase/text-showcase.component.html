typography

A typography é um componente de design que se refere à forma como o texto é apresentado em uma interface de usuário. Ela inclui características como:

  • Fonte (familia, tamanho, estilo)
  • Cor
  • Alinhamento
  • Espaçamento entre linhas
  • Tamanho de parágrafo
